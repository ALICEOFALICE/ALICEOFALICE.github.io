如何在服务器上部署一个HEXO

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

之所以入坑HEXO是因为看到了知乎上几个大佬的推荐，在加上想试试CDN（老的博客只能用OSS，不能用CDN），索性就部署一个HEXO博客系统。

介绍

HEXO采用全静态（应该是？），可以生成网页后直接push到Github，在通过访问的方式读取（再也不用担心被打了）。

部署准备

  1. HEXO使用了node.js框架进行搭建,因此,你必须要安装node.js,关于如何安装node.js,可以参考这篇文章,再次不进行赘述.
  2. git请使用yum install git 进行安装
